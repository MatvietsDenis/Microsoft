Основні дати в історії компанії

1975

Друзі по навчвнню Біл Гейтс та Пол Ален
заснували компанію Microsoft.
Першим їх продуктом
був інтерпритатор мови Bsic.

1980

Microsoft підписала договір з IBM
на розборку операційнрї системи
MS-DOS.

1983

Реліз першої версії MS World

1985

Реліз першої версії Excel. Версія була
призначена тільки для компютера
Macintosh.

1995

Реліз Windows 95 з оновленним
інтерфейсом на кнопку Пуск

1985

Біл Гейтс став найбагатшою людиною
у світі та збкрігав цей статус до
2017 року
